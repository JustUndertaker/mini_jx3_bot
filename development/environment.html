# Python环境

python环境

本项目使用python版本为3.10.5，请确保你的环境大于python3.10。

# Nonebot2版本

Nonebot2版本

本项目使用Nonebot2的版本为beta4，请确保环境在beta4+。

# 使用虚拟环境

虚拟环境

养成良好的习惯，不是吗？强烈推荐你使用conda管理环境，你只需要使用以下命令：

conda create -n env_name python=3.10
conda activate env_name
