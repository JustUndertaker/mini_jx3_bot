# 项目结构

当前项目结构如下：

📦mini_jx3_bot
 ┣ 📂data                   # 运行数据目录
 ┣ 📂docs                   # 文档
 ┣ 📂logs                   # 运行日志目录
 ┃ ┣ 📂debug
 ┃ ┣ 📂error
 ┃ ┗ 📂info
 ┣ 📂src                    # 代码目录
 ┃ ┣ 📂internal             # 插件公用模块目录
 ┃ ┣ 📂managers             # 管理插件目录
 ┃ ┣ 📂modules              # 数据库模型目录
 ┃ ┣ 📂plugins              # 自定义插件目录
 ┃ ┗ 📂utils                # 工具模块
 ┣ 📂template               # 渲染模板目录
 ┣ 📜.env.prod              # 项目环境文件
 ┣ 📜bot.py                 # 项目入口
 ┗ 📜requirements.txt       # 项目依赖
